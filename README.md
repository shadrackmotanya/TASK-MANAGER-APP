# NovaTech Hub - Task Manager App

A simple and interactive Task Manager web app built using **HTML, CSS, and JavaScript**. It helps users manage daily tasks efficiently with a clean and modern interface.

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Filter tasks (All / Completed / Pending)
* Saves tasks automatically (local storage)

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation & LocalStorage)

## 📂 How to Run

1. Download or clone the project
2. Open `index.html` in your browser

## 💡 Usage

* Type a task and click **Add**
* Click ✔ to mark as complete
* Click ✖ to delete
* Use filter buttons to organize tasks
